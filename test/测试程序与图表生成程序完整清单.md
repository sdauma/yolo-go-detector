## 6. 测试执行顺序建议

1. **环境准备**：确保关闭Turbo Boost并设置高性能电源模式
2. **强化性能测试**：
   - Python大模型：`python python_reinforced_benchmark.py`
   - Go大模型：`go run go_reinforced_benchmark.go`
   - Python轻模型：`python python_reinforced_benchmark_small.py`
   - Go轻模型：`go run go_reinforced_benchmark_small.go`
3. **冷启动测试**：
   - Python：`python python_cold_start_decomposition.py`
   - Go：`go run go_cold_start_decomposition.go`
4. **内存测试**：
   - Python：`python python_memory_standardization.py`
   - Go：`go run go_memory_standardization.go`
5. **输出一致性测试**：
   - Python：`python python_output_consistency.py`
   - Go：`go run go_output_consistency.go`
6. **数据分析**：`python data_analysis.py`
7. **生成图表**：`python generate_all_charts.py`

## 7. 图表生成与使用

所有图表生成程序都将图表保存到 `results/charts/` 目录，同时生成PNG和PDF格式：

- **PNG格式**：600dpi，用于论文引用、网页展示
- **PDF格式**：矢量图，用于期刊投稿、高质量打印

论文直接引用 `results/charts/` 目录下的PNG图表，无需复制。
